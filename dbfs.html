Databricks File System (DBFS) commands

# Listing files 
%fs ls dbfs:/mnt/

# Copying a file
%fs cp dbfs:/mnt/source.csv dbfs:/mnt/destination.csv

# Removing a file
%fs rm dbfs:/mnt/temp_file.csv

# Checking version (SH- run shell)
%sh python --version

# Running another notebook
%run /Users/my_user/my_other_notebook

# using markdown for documenting
%md

# Using pip to install
%pip install pandas
